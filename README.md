# 📊 Sales Analysis System

A Python-based Sales Analysis System that analyzes product sales data, calculates revenue, generates reports, and creates visualizations using Pandas and Matplotlib.

---

## 🚀 Features

- Product-wise Revenue Analysis
- Category-wise Revenue Analysis
- Best Selling Product Identification
- Highest Revenue Product Analysis
- Lowest Revenue Product Analysis
- Top 3 Products Report
- CSV Report Export
- Revenue Bar Chart Generation
- Revenue Pie Chart Generation
- Business Performance Summary

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib

---

## 📂 Project Structure

```text
SalesAnalysisSystem/
│
├── sales_analysis.py
├── sales_report.csv
├── revenue_bar_chart.png
├── revenue_pie_chart.png
└── README.md
```

---

## 📊 Dataset Fields

| Column | Description |
|----------|------------|
| Product | Product Name |
| Category | Product Category |
| Quantity | Units Sold |
| Price | Unit Price |
| Revenue | Quantity × Price |

## ▶️ How to Run

1. Install required libraries:

```bash
pip install pandas matplotlib
```

2. Run the project:

```bash
python sales_analysis.py
```
